About the App

Unlock your perfect park run adventure with the parkrunner tourist App! This essential tool lets you easily find park run events worldwide and discover nearby hotels, all from a user-friendly map interface.

Features

  • Easy to Use
  • Find Best Price Hotels/Campsites
  • Get Directions to the Event
  • Quickly Check the Weather Forecast for the Event
  • Shows Event Cancellations
  • Instantly Check the Volunteer Roster and Course Map

Why Use parkrunner tourist?

Exploring new locations? Our app ensures you can plan your runs and accommodations with ease. Track your park run journey, find the best places to stay, and make your next run unforgettable. Download the parkrunner tourist App today and start your park run journey!
